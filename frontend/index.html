Know if an image has been altered

PixelProof analyzes images to detect signs of editing, recompression, or authenticity — instantly.

How it works

PixelProof analyzes compression patterns and metadata inside an image.

  • Metadata (EXIF) reveals how and when the image was created
  • Compression analysis detects inconsistencies introduced during editing or saving
  • Overlay visualization highlights areas with unusual differences

Brighter areas in the overlay indicate stronger compression differences.

Results are based on statistical image analysis and should be interpreted as indicators, not absolute proof.
